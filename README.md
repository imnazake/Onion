# Onion
A modern C++ application template utilizing Dear ImGui and DirectX for immediate-mode graphical user interfaces.

## ‚ú® Key Highlights

* **Custom UI Engine:** Modern "Pill-style" geometry with `12.0f` rounding for a premium macOS/Windows 11 feel.
* **Google Material Palette:** Optimized dark-mode surfaces (`#121212`) with high-contrast **Electric Blue** interactive states.
* **Embedded Typography:** Integrated **Rubik** font (via `Font.h` byte array) for a sleek, modern look without external dependencies.
* **Auto-Layout Ready:** Configured for seamless window auto-sizing and pixel-perfect padding.
* **DirectX Backend:** Low latency rendering suitable for tools, overlays, or dashboards.
